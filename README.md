# Resolva Jato

Projeto estático inicial para um catálogo premium de recursos gratuitos e validados.

## Estrutura

- `index.html`: página única com header, busca, filtros, grid e CTAs.
- `src/resources.js`: base de dados inicial dos recursos.
- `src/app.js`: busca, filtro por categoria, contagem e modo claro/escuro.
- `src/styles.css`: interface utilitária, responsiva e sem dependências de build.

## Como visualizar

Abra `index.html` no navegador. Como o projeto usa módulos JavaScript, se o navegador bloquear a abertura direta, sirva a pasta com um servidor local simples.
